Projeto Integrado (3º Semestre - 01/2024)



Projeto do 3º semestre do curso de Desenvolvimento de Software Multiplataforma da FATEC de São José dos Campos – SP.

Este projeto teve como objetivo a criação de uma aplicação web que visa facilitar o manuseio dos ativos de uma empresa, sendo acompanhar quem atualmente possui o ativo, acompanhar manutenções do ativo, entre outros.

Atuei como Product Owner, intermediando a comunicação entre o cliente e o time de desenvolvimento. Nessa função, elaborei a documentação necessária, tais como: backlogs do produto e das sprints, DOR, DOD, e realizei os testes de aceitação.

Além disso, fiz parte do desenvolvimento do projeto, colaborando com a criação dos componentens de notificação e das interfaces de manutenção, histórico do ativo e relatórios.

Tecnologias utilizadas: Figma, Git, GitHub, TypeScript, Java, Jira, PostgreSQL, Supabase, React, Spring.

Soft Skills desenvolvidas: Trabalho em Equipe, persistência, resiliência e designação de responsabilidades.

Hard Skills adquiridas: Adquiri conhecimentos em backend (Java), aprimorei meus conhecimentos em frontend (React, TypeScript e bibliotecas), aprendi a usar o PostgreSQL e o Supabase.
